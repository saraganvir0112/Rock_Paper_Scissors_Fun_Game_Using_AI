# Rock Paper Scissors Fun Game Using AI 🤖✊

A real-time Rock Paper Scissors game where you play against AI using your hand gestures and webcam — no keyboard or mouse needed!

## 🎮 How to Play
- ✊ Make a **Fist** = Rock
- 🖐️ Open your **Full Hand** = Paper
- ✌️ Show **Two Fingers** = Scissors
- Press **Q** to quit the game

## 🛠️ Technologies Used
- Python 3.11
- OpenCV — camera and display
- MediaPipe — hand tracking
- NumPy — image processing

## ⚙️ Installation

1. Clone the repository
```bash
git clone https://github.com/saraganvir0112/Rock_Paper_Scissors_Fun_Game_Using_AI.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the game
```bash
py -3.11 RPS.py
```

## 📸 Features
- Real-time hand gesture detection
- AI makes random moves
- Live score tracking
- 3 second timer between rounds
